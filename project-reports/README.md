## Technology Overview

The portfolio spans a broad range of languages, frameworks, and paradigms:

- **Languages**: C, C++ (17/20/23), Rust, Kotlin, Java, TypeScript, PHP, JavaScript, GLSL
- **Frontend**: Qt 5/6 (Widgets), Jetpack Compose, Iced (Rust), Angular 16, SvelteKit, Three.js/WebGL, Java Swing
- **Backend**: PHP, Node.js, Java Servlets, Python
- **Databases**: PostgreSQL, MongoDB (NoSQL)
- **Distributed Systems**: MPI, Sockets TCP, rclone
- **Mobile**: Android (Kotlin), iOS/macOS (Swift/SwiftUI)
- **Build & CI**: CMake, Conan 2, Gradle, pnpm, GitHub Actions
- **Cryptography**: RSA, modular arithmetic, primality testing
- **Multimedia**: FFmpeg, OCR, AI transcription
